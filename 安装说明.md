# DoriCall 安装说明 🍎

办公室桌面点名提醒:点一下同事名字,TA 屏幕右上角弹出苹果人卡片 + 提示音,点「收到」你这边马上知道。不用服务器、不用登录,同一个 Wi-Fi 就能用。

## 安装(1 分钟)

1. 下载 `DoriCall.zip`,双击解压,把 **DoriCall** 拖进「应用程序」文件夹
2. **右键点击 DoriCall → 打开 → 再点「打开」**
   (内部工具没有上 App Store 公证,第一次必须右键打开,以后就正常了)
   - 如果还是被拦:打开 系统设置 → 隐私与安全性 → 拉到底部点「仍要打开」
3. 首次启动会问「你是谁?」——**选自己的名字**(只选这一次)
4. 如果弹出「DoriCall 想要查找并连接本地网络设备」→ 点 **允许**(必须,不然发现不了同事)

装好后菜单栏会出现一个  图标,开机自动启动,不用管它。

## 使用(30 秒上手)

点菜单栏  图标:

| 操作 | 效果 |
|---|---|
| 点人名旁的 **🔔** | TA 屏幕弹「xx 在叫你」卡片 + 提示音 |
| 点人名旁的 **👍** | 给 TA 无声点个赞 |
| **点人名本身** | 出现输入框,可以发一句短话(回车发送) |
| **📢 叫所有人** | 给所有在线同事广播 |
| **🔕 勿扰** | 卡片静默堆着不响,对方自动收到「勿扰中」 |

收到卡片时:点 **「收到 👌」** 或 **「等会 🫷」**,对方立刻看到你的回应。

- 名字旁的彩色圆点 = 在线;灰点 + 「离线」= TA 没开机或不在同一网络
- 有没处理的呼叫时,菜单栏图标会闪铃铛

## 常见问题

- **看不到同事在线?** 确认双方在同一个办公室 Wi-Fi;第 4 步的本地网络权限点了「允许」(补救:系统设置 → 隐私与安全性 → 本地网络 → 打开 DoriCall)
- **开着 Clash 等代理有影响吗?** 没有,DoriCall 局域网直连,不走代理
- **想换人/选错名字了?** 退出 DoriCall(菜单 → 退出),终端执行 `defaults delete life.dori.doricall whoami` 再打开重选
- **卸载**:菜单 → 退出,然后把 DoriCall 从「应用程序」删掉

## 系统要求

macOS 13 (Ventura) 及以上。
